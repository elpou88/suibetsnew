After Downloading

  1. Extract the archive with: 
    tar -xzvf suibets-platform.tar.gz
  2. Navigate to the extracted directory: 
    cd suibets-platform
  3. Copy the example environment file: 
    cp .env.example .env
  4. Edit the .env file with your API keys
  5. Install dependencies: 
    npm install
  6. Start the application: 
    npm run dev

Note: DATABASE_URL is optional. The application can run without a database connection by using blockchain storage as a fallback.
